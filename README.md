# Lucky Neko & Friends | Premium Kawaii Experience

A stunning, award-winning aesthetic web design featuring Sanrio-style characters and a premium interactive experience. This project combines modern web technologies with vibrant, "kawaii" aesthetics to create a unique and engaging landing page.

![Lucky Neko](images/cat_main.png)

## âœ¨ Features

- **Premium Design**: Vibrant color palettes (kpink, kpurple, kgold) and modern typography (Outfit, M PLUS Rounded 1c).
- **Interactive Background**: A 3D background powered by **Three.js** featuring floating gold coins.
- **Custom Cursor & Effects**: An interactive cursor with custom trailing particles and flower explosions on click.
- **GSAP Animations**: Fluid entry animations and scroll-triggered effects.
- **Responsive Layout**: Built with **Tailwind CSS** for a seamless experience across all devices.
- **Character Showcase**: A grid of lucky friends with interactive hover states and detailed modal views.
- **SEO Optimized**: Includes meta tags for search engines and social media previews (Open Graph & Twitter Cards).
